# 📓 DEVELOPMENT_NOTES Template

Template untuk mencatat method kerja, asumsi, hasil debug, dan keputusan teknis. Ini semacam buku harian bengkel repo. 🧰🛠️

```markdown

- ...

- ...

- ...

### 📁 Files Changed
- ...

### 🧪 Validation
- ...

### ✅ Result
- ...

### ⚠️ Remaining Risks
- ...

### 🚀 Next Steps
- ...
```

## ✅ Rules

- Catat asumsi sebelum implementasi.
- Catat command yang benar-benar dijalankan.
- Jangan menulis secret asli.
- Kalau ada error yang belum selesai, tulis jujur di Remaining Risks.
