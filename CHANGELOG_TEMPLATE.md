# 📝 CHANGELOG Template

Gunakan format ini untuk mencatat semua perubahan penting. Cocok untuk tiap repo target. 📚✨

```markdown
# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- ✨ ...

### Changed
- 🔁 ...

### Fixed
- 🐞 ...

### Security
- 🔐 ...

### Docs
- 📚 ...

### Removed
- 🗑️ ...

## [0.1.0] - YYYY-MM-DD

### Added
- 🎉 Initial release.
```

## ✅ Tips

- Satu sesi agent = minimal satu entry changelog.
- Pisahkan bug fix dan security fix.
- Jangan tulis secret/token/API key.
- Pakai tanggal jelas supaya histori repo tidak jadi kabut digital. 🌫️
