# 🗺️ ROADMAP Template

Template roadmap untuk mengubah repo dari “jalan tapi belum jelas arah” menjadi “punya jalur, rambu, dan checkpoint.” 🚦✨

```markdown
# Roadmap

## P0 🔥 Critical Stability
- [ ] Fix build error
- [ ] Fix runtime crash
- [ ] Fix security blocker
- [ ] Add missing environment example

## P1 🚀 Launch Ready
- [ ] Improve README
- [ ] Add deployment guide
- [ ] Add health check
- [ ] Add error/loading/empty states
- [ ] Add basic tests

## P2 ✨ Product Features
- [ ] Add dashboard
- [ ] Add export feature
- [ ] Add settings page
- [ ] Add user-friendly logs
- [ ] Add analytics summary

## P3 🧱 Scale & Automation
- [ ] Add CI/CD
- [ ] Add monitoring
- [ ] Add database adapter
- [ ] Add Docker support
- [ ] Add rate limiting
```

## 🧭 Per Item Format

Gunakan format ini kalau mau roadmap lebih detail:

```markdown
## Feature: [Nama fitur]

### Problem
- ...

### User Value
- ...

### Technical Approach
- ...

### Affected Files
- ...

### Complexity
Low / Medium / High

### Risk
- ...

### Test Plan
- ...

### Acceptance Criteria
- [ ] ...
```
